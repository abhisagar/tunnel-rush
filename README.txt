Instructions to run:
	firefox index.html (open the index.html file using your browser)

Controlls:
	Left-arrow 		- tunnel will rotate left
	Right-arrow 	- tunnel will rotate left
	t				- switch between texture and color
	p 				- pause
	space			- jump

Description:
	The player will move with certain speed and will face many obstacles. Gradually the speed will increase.
	Score will increase on passing different levels.
	if you hit anything then game is over.
